# Paradise Motorsports - Racing Game in Unity 3D

A single player racing game with multiplayer functionality, built in Unity 3D with C#,
developed as a Bachelor of Engineering final year project and documented in an IEEE
style journal report.

> **Note on dates:** This project was originally completed in 2017 as a final year
> Bachelor of Engineering (Computer Engineering) project. It was uploaded to GitHub in
> June 2026 after my previous GitHub account was deleted. Commit dates are set to reflect
> the original completion date.

## Overview

Paradise Motorsports is an entertaining racing game built for the Windows platform. The
player must cross a path full of obstacles in the shortest time possible, with difficulty
increasing gradually as the game progresses. The game is a single player experience that
also provides multiplayer functionality, letting users race against friends. A time
attack mode lets players track and beat their own high scores.

The emphasis is on responsive timing and speed, organized level design, and visually
appealing graphics to give players the best possible experience.

## Features

- Single player racing with obstacle courses
- Multiplayer races against friends
- Time attack mode with personal high score tracking
- Progressive difficulty across levels

## Tech Stack

Unity 3D game engine, C#, Windows platform.
